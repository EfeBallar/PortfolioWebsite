# Personal Portfolio Website

Welcome to my portfolio website!

## Features

- **Landing Animation**: White screen with "Hey!" texts in different languages that roll up to reveal the main content
- **About Section**: Interactive timeline and rotating fun facts
- **Skills Section**: Organized skill categories with hover effects
- **Projects Section**: Card gallery with hover overlays and tech stack display

## Sections

1. **Landing Page**: Initial white screen with animated "Hey!" text
2. **About Me**: Personal story with interactive timeline and fun facts
3. **Skills**: Grid of technical skills organized by category
4. **Projects**: Portfolio of featured projects with source code
5. **Where I've Been**: Map of my biography!

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
